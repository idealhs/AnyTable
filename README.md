Enhance how you work with web tables

## Overview

AnyTable is a lightweight, clean, high-performance browser extension. It brings sorting, filtering, statistics, and CSV export directly into the page, so you can process data quickly right in the browser.

## What You Can Do with AnyTable

- Fast and simple sorting and filtering.
- Use advanced sorting for multi-column priority sorting, with support for rich unit-based values and automatic type detection.
- Use advanced filtering to build nested condition groups for complex logic across multiple columns.
- Generate statistics rows with count, sum, average, median, variance, and more.
- Export the current table view to CSV.
- Watch for page changes and automatically enhance tables that appear later.
- Support many non-standard tables, including merged cells, wide headers, nested tables, and more.

## Get AnyTable

| Channel | Best for | Install |
| --- | --- | --- |
| Firefox | Direct install from the official Firefox store | [Firefox Add-ons](https://addons.mozilla.org/en-US/firefox/addon/anytable) |
| Chrome | Direct install from the official Chrome store | [Chrome Web Store](https://chromewebstore.google.com/detail/anytable/dbadglamnjdkmadaocimiijibamaghkk) |
| Edge / Brave / Vivaldi / Arc | Most Chromium-based browsers | Try installing from the Chrome Web Store first |
| Manual package | Browsers that do not support direct store install, offline install, or manual deployment | [GitHub Releases](https://github.com/idealhs/AnyTable/releases) |

### Install on Other Browsers

For most Chromium-based browsers, AnyTable usually works as well. You can try installing it from the [Chrome Web Store](https://chromewebstore.google.com/detail/anytable/dbadglamnjdkmadaocimiijibamaghkk).

1. If direct installation is not available, go to [GitHub Releases](https://github.com/idealhs/AnyTable/releases) and download the latest release package.
2. Extract the downloaded ZIP file.
3. Open your browser's extension management page and enable Developer mode.
4. Choose `Load unpacked`, `Install unpacked`, or the equivalent option in your browser.
5. Select the extracted extension folder to complete the installation.
